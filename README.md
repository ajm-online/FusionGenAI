# FusionGenAI: RAG + Multimodal AI + GANs

Advanced GenAI project integrating:
- Text-based RAG
- Multimodal AI (text + image)
- Simple GAN-based image generation
- Optional web app interface

## Setup
1. Clone repo or download files.
2. Install dependencies:
   pip install -r requirements.txt
3. Run notebooks or Flask app:
   python web_app/app.py
